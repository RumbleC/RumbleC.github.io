Java面试题目综合汇总

发表于2021-10-19|更新于2021-11-03|Java面试题目
|阅读量:

基础

https://mp.weixin.qq.com/s/5h4exPg-xWW6sqv3rGxL-Q

  1. synchronized怎么实现的？锁升级过程

  2. ReentrantLock怎么实现的？相比于synchronized有哪些特征，可重入，非公平竞争是怎么实现的。

  3. 线程池（ThreadPoolExecutor）的七个参数都是什么阶段使用到？流程图

  4. 线程池可用队列类型有哪些？拒绝策略有哪些？

  5. GC算法有哪些，各自的优缺点是什么

  6. cms的执行四个阶段，每个阶段做了什么事情。年轻代回收怎么触发的老年代回收(卡表是怎么新建、更新引用关系的)。

  7. new object创建一个对象做了那些事情(1、查询这个对象是否已经创建，2、分配内存3、初始化)第二个阶段分配内存是怎么保证多线程安全的

  8. 创建对象完了以后怎么保证多线程读数据的一致性(多核CPU缓存一致性协议)

  9. 数据库隔离级别介绍。

  10. RR隔离级别下使用间隙锁真的解决了幻读问题吗？

  11. 不管是主键还是普通索引，两个线程，一个线程先select，第二个线程insert，然后第一个线程再select, select出来几条？一个线程先select，第二个线程insert，然后第一个线程再update, update影响得记录条数是几条？(MVCC)

  12. start Transaction，Transaction底层是怎么实现的。

  13. Spring事务是怎么实现的(动态代理)，事务传播机制。

  14. 事务失效的情况，为什么会失效(也就是动态代理不起作用的情况)。

阿里真实面试题

https://mp.weixin.qq.com/s/jb1Afc6gxTdo7KkyWxcw9Q

  1. ConcurrentHashMap，HashMap原理，1.7和1.8的区别。
  2. 怎么设计一个秒杀系统，瞬时高并发设计，限流设计。
  3. 分布式锁Redisson加锁原理，锁失败，锁续期，公平锁，非公平锁，集群模式下有哪些问题。
  4. Redis阻塞队列。
  5. Java中锁有哪些，锁升级过程，锁原理。
  6. spring bean处理器，生命周期，bean循环依赖。
  7. jvm年轻代，老年代垃圾收集算法，对象何时进入到老年代。
  8. 怎么排查生产环境问题。
  9. 分库分表。
  10. redis多个服务器自增怎么区分。
  11. 项目中觉得有亮点的地方。
  12. redis单线程效率高的原因。
  13. redis多路复用模型，执行流程。
  14. netty的reactor模型。
  15. MySQL分库分表
  16. MySQL所以的优化，隔离级别，mvcc，redo和binlog写入时机，同步机制。
  17. jvm优化经历

综合面试题

https://mp.weixin.qq.com/s/gllKJGLH0nN5DbCTIBqwKA

八股文

  1. 过往项目经历，项目亮点，遇到什么难题，怎么解决的，有没有其他解决方案。讲讲项目架构以及做了哪些高可用方案设计
  2. MQ( ROCKETMQ)项目中使用场景，ROCKETMQ架构原理
  3. ROCKETMQ是怎么保证消息不丢失的，业务中消息消费幂等性是怎么保证的，场景。
  4. REDIS做缓存怎么保证缓存不被穿透、击穿。
  5. 怎么防止商品超卖，阐述处理流程。
  6. 数据库存储原理，做过哪些优化，怎么优化、索引合并、最左匹配原则，mvcc。。
  7. 数据库ICP优化
  8. JVM使用的那种垃圾收集器，分别介绍CMS和G1对比，优缺点，适用场景。
  9. 介绍跨代怎么回收，卡表、RSET结构，添加、更新引用时机。
  10. MySQL如果写入QPS过高怎么优化(体现思路【分库分表->缓冲区参数优化 由大到小】)。
  11. MySQL如果读QPS过高怎么优化(同上)。

算法

  1. 给定一个数组，求最大连续子序列之和
  2. 给定一个二叉树，求二叉树中序遍历(非递归实现)

高级开发面试题目

https://mp.weixin.qq.com/s/P4O6LvAFLS1FwwVenKvX5Q

Redis基础

  • zSet的使用(怎么实现类似MySQ Llimit功能)

  • incr是先加后获取值还是先获取值然后执行加1

  • Redis数据结构实现原理

  • Redis单线程模型分析

MySQL基础

  • B+树基本存储结构

  • B+树为什么非叶子结点不存储数据

  • 联合索引的最左匹配原则(各种场景举例判断是否走索引)

  • MySQL8使用索引的index merge特性

  • SQL一般优化方法（索引的使用、索引失效、explain的使用-哪些场景下看哪些字段判断需要进行SQL优化和SQL优化的入手点）

  • MySQL锁相关-间隙锁（gap-lock）

Java SE基础

  1. forkJoinPool的原理

  2. forkJoinPool的思想(分而治之)以及分而治之适合、不适合哪些场景

  3. 函数闭包

  4. Automic包下类实现原理(CAS、UNSAFE、缓存一致性协议)

框架相关

  1. mybatis Executor除了BATCH还有哪些类型，BATCH有没有其他替代方式

  2. mybatis怎么控制事务回滚、提交

  3. spring事务传播机制原理

  4. IOC、AOP原理
